# 📚 Assignment Service Web Application

A full-stack web platform that connects students with assignment solutions through an efficient submission and delivery system.

## 🌐 Live Is Live
**[View Live Application](https://ashen-assignment-app-fadpbtfxduatavf7.eastasia-01.azurewebsites.net/)**

## 📋 Overview
This assignment service platform streamlines the process of submitting academic assignments and receiving professional solutions. The application features a dual-interface system with separate portals for users and administrators, ensuring smooth workflow management from submission to payment.

## ✨ Key Features

### User Portal
- **Assignment Submission** - Easy-to-use interface for submitting assignment details and requirements
- **Email Notifications** - Automated email updates for submission confirmations and solution delivery
- **Dashboard Notifications** - Real-time inbox notifications within the user dashboard
- **Secure Payment Gateway** - Integrated payment processing for completed assignments
- **Solution Access** - Convenient access to delivered solutions through the dashboard

### Admin Panel
- **Assignment Management** - Centralized dashboard for receiving and tracking user submissions
- **Email Integration** - Automated email notifications for new assignment requests
- **Approval Workflow** - Review and approve assignments before solution delivery
- **Solution Distribution** - Direct solution delivery to users via email and dashboard

## 🛠️ Tech Stack

- **Backend:** Java with Maven
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** Microsoft SQL Server
- **Deployment:** Azure App Service

## 🔐 Security Features

- Enhanced database schema with secure user credential storage
- Protected user authentication and authorization
- Secure payment gateway integration
- Data validation and sanitization

## 🗄️ Database Architecture

The application utilizes a well-structured MS SQL database schema featuring:
- User management tables with encrypted credentials
- Assignment tracking and status management
- Notification and messaging system tables
- Payment transaction records
- Admin activity logs

## 🚀 Workflow

1. **User Registration/Login** - Secure account creation and authentication
2. **Assignment Submission** - Users submit assignment details through the form
3. **Admin Notification** - Administrators receive email alerts for new submissions
4. **Review & Approval** - Admin reviews requirements and approves the assignment
5. **Solution Delivery** - Completed solutions sent via email and dashboard
6. **Payment Processing** - User proceeds to secure payment gateway
7. **Confirmation** - Final confirmation and solution access granted
